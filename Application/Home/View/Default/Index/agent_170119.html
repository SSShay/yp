三大难题压头顶


您是否遇到了这些问题？

目前在中国的家庭里，孩子的事情是头等重要的事情，孩子的健康、安全、教育等问题是家长们的核心关注。孩子每天是否按时到校，放学后是否按时回家，孩子在学校的表现等等都牵挂着父母的心。

教育信息化建设

教育信息化“十三五”规划提出到2020年基本建成“三通两平台”，许多学校由于教学条件的限制，其“三通两平台”的建设显得举步维艰。

家校沟通问题

传统家访、家长会无法把学校与家长紧密地联系在一起，家长不能全面了解孩子在校学习情况，老师无法掌握学生在家学习进度，老师与家长之间不能取得及时的沟通，导致家校沟通障碍问题。

校园安全问题

近年来儿童上下学走失案件频发，教育部门、学校、家长高度关注校园安全问题，如何保障儿童上下学安全成为社会关注的热点。

学生安全

即时考勤

家长、学生进出校门，其他亲属和老师可实时接收到学生接送/考勤情况；老师可实时查询学生考勤情况，实现学生出勤智能化管理。

精准定位

家长通过手机APP查询学生位置，随时随地了解孩子去向,家人安全一手掌控。

家校沟通

公告通知

教师一对多发送，高效传达通知公告，省时省力；学校自定义审核权限，有效规避不良信息传播；重要通知可选择“需要家长确认”选项，接收效果教师可通过确认反馈列表了解。

家校联络

家长与老师一对一的沟通交流，让家长老师及时了解学生在校在家情况，达到家校共育目的，为学生健康成长提供有效保障。

学习管理

成绩管理

家长可在线查询学生成绩情况，通过成绩曲线对学生进行针对性指导；老师可在线查询全班的成绩情况，通过集体和个人成绩情况，有针对性的制定教学计划，实现教学的智能化数据分析。

作业发布

家长可在线查询学生每天作业情况，对学生完成情况进行指导；老师可通过手机端、PC端发布作业，实现作业发布的智能化管理。

智能校园

校园一卡通

校园一卡通旨在构建便捷、高效的校园智能化服务平台，利用一卡通平台系统，整合校内身份认证、门禁控制、考勤记录、金融消费等硬件设备，将一卡通系统与身份识别、考勤、门禁、消费等终端设备深度整合。

录播系统

教师通过全自动录播系统录制课程，学生远程学习，远程互动教学等等，可以随时随地进行观摩、学习，不再受空间和时间的限制。
