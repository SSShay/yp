远磐概述

杭州远磐互联科技有限公司（简称远磐科技）是经杭州市滨江区政府批准入驻智慧E谷，围绕国家教育信息化、“三通两平台” 落地开发的专业软件服务商，是国家大力扶持的智慧教育产业的领先单位，也是国内最具影响力的教育信息化企业之一。
远磐科技秉承“关爱学生安全成长”为宗旨，依托强大的技术核心资源，不断完善产品体系，扩大服务范围，形成以 微家校APP和电子学生证为核心的“微家校”服务项目。截止2016年6月，微家校项目已覆盖全国十余个省份，签约了4000多 所学校，40多万人入驻微家校APP。同时与中国移动、中国联通两大通信服务商建立长期战略合作伙伴关系，让40余万家长 不再为孩子的出行安全担忧；让10000多名老师实现轻松管理学生；让20余万留守儿童得到来自父母的关爱。
远磐科技致力于解决中小学教育问题，不断为广大学生家长提供最优质的服务，让家长放心，让老师安心，让学校开心！
