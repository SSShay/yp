贵州省级代理

代理商：赵总
代理区域：贵州省代
代理性质：省级代理
签约学校：2500所
销售额：3000余万
回报周期：4个月

贵州地区高原山地居多，孩子放学后，学校没有能力护送孩子安全到家，家长也关心孩子路上的安全。贵州分公司和 贵州联通及时推出的微家校系统，主要面向贵州省所有1000万左右中小学生、家长、老师，提供家校互动服务和智能 设备定位考勤服务，帮助学校和家长解决了共同关注的孩子安全到家问题。自2015年9月份上线以来，目前有近10万在 线活跃用户，智能设备下发1万多台，设备还在陆续下发到签约学校的家长和孩子手里。

宁夏总代理

代理商：宁夏移动分公司
代理区域：宁夏
签约学校：2000所
销售额：1500万元
回报周期：6个月

宁夏地区，原来大部分学校是在使用宁夏移动推出的校讯通业务，学校主要通过短信方式和家长沟通，了解孩子在校 情况和布置作业等。校讯通是个功能单一、业务过时的产品，不能满足宁夏地区学校教育信息化的增长需求了。合作 推出的宁夏移动人人通，主要面向宁夏自治区所有80万中小学生、家长、老师，提供家校互动服务和智能设备定位考勤 服务，及时得解决了相关问题。自2015年9月份上线以来，目前有近10万在线活跃用户（包括5万左右套餐订购用户）。

河南省代理
